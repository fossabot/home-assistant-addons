# Profilarr Documentation

## Overview

Profilarr is a profile management tool for *arr applications (Sonarr, Radarr, Lidarr, etc.). It helps you manage and synchronize quality profiles, custom formats, and other settings across your *arr instances.

## Installation

1. Navigate to the Supervisor panel in Home Assistant
2. Click on the Add-on Store tab
3. Add this repository if not already added
4. Find "Profilarr" in the list
5. Click on the add-on and then click "Install"

## Configuration

| Option | Description |
|--------|-------------|
| `log_level` | Controls logging verbosity (trace, debug, info, notice, warning, error, fatal) |
| `tz` | Timezone for the add-on (e.g., UTC, America/New_York) |

## Usage

After starting the add-on:

1. Click "Open Web UI" to access the Profilarr interface
2. Configure your *arr application connections
3. Import or create quality profiles
4. Sync profiles across your instances

## Features

- Manage quality profiles for Sonarr, Radarr, and other *arr apps
- Import/export profile configurations
- Sync settings across multiple instances
- Custom format management
