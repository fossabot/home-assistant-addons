# Profilarr

Profile management for *arr applications (Sonarr, Radarr, etc.)

## About

Profilarr helps you manage and synchronize quality profiles, custom formats, and other settings across your *arr instances.

## Installation

1. Add this repository to your Home Assistant add-on store
2. Install the add-on
3. Configure the add-on (if needed)
4. Start the add-on

## Configuration

| Option | Description |
|--------|-------------|
| `log_level` | Controls logging verbosity |
| `tz` | Timezone for the add-on |
